Adding The Image

The base code in Material.io doesn't come with a section for images, so you have to add that in HTML and add custom CSS for it.


            
          

#card-example .mdc-card__media {
  background-image: url("https://www.premera.com/images/Style_Guide/Jumbotron_Woman_650x275.jpg");
  background-size: cover;
  background-repeat: no-repeat;
  height: 200px; /* Note: Height works in conjunction with height of card */
}

Custom Sizing

Cards don’t come with a predefined width or height. They expand horizontally to fill all available space, and vertically to fit their contents. If you’d like to maintain a consistent height and width across cards, you’ll need to set it in your styles. You might need to muck with the media height and the card height/width to make sure you don't have any weird gaps.


#card-example.mdc-card {
  height: 450px; /* Note: Height works in conjunction with height of media below */
  width: 350px;
}

Grid Lists

MDC Grid List provides a RTL-aware Material Design Grid list component adhering to the Material Design Grid list spec. Grid Lists are best suited for presenting homogeneous data, typically images. Each item in a grid list is called a tile. Tiles maintain consistent width, height, and padding across screen sizes.

For more details and information on how to tweak this layout, review the Material Web Components guidelines.
